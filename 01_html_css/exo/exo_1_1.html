C - direction

flexbox nous permet d'agencer ces éléments dans le sen que l'on veut.Avec flex-direction, oon peut les positionner verticalement ou encore les inverser. il peit prendre les valeurs suivants:
-> row : origanisés sur une ligne (par défaut)
-> column : organisés sur une colonne
-> row-reverse : organisé sur une ligne mais en ordre inversé
-> column-reverse : organisé sur une colonne, mais en ordre inversé
