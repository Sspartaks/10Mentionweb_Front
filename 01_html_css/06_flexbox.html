D - Alignement

Les éléments sont organisés soit horizontalement (par défaut), soit verticalement.

  • si vos éléments sont organisés horizontalement : l'axe principale (main axis) est l'axe horizontale, l'axe secondaire (cross axis) est l'axe vertical.
  • si vos éléments sont organisés verticalement : l'axe principale (main axis) est l'axe verticale, l'axe secondaire (cross axis) est l'axe horizontale.

1 - alignement sur l'axe principal

afin d'aligner nos éléments sur l'axe principal : on utilise la propriété du flex justify-content

cette propriété gére la distribution des emplacements des flex-items suivant la place disponible dans le flex-container.Elle le fait , selon le positionnenemnt eda l'axe principal

  • flex-start : aligner au début (par défaut)

  • space-between : les éléments sont étirés sur tout l'axe (il y a de l'espace entre eux)
  • space-around : les élément sont étirés sur tout l'axe, mais ils laissent aussi de l'espace sur les extrémités
  • space-evenly : les élément sont répartis uniformément dans le conteneur d'alignememnt le long de l'axe principal

F - Répartir plusieurs lignes

Si vous avez plusieurs ligne dans votre flexbox,vous pouvez choisir comment celles-ci seront réparties avec align-content. Cette propriété permet de gérer l'alignement des différents lignes (ou colonnes) les par rapports au autres. Cette propriété n'aura d'éffet que si le conteneur flexible posséde plusieurs lignes(si l'axe principale est horizontal) ou plusieurs colonnes ( si l'axe principle est vertical) et que si le flex-wrap est présent.

Les valeurs de cette propriété :

  • flex-start : les éléments sont placés au début
  • flex-end : les élélements sont placés à la fin
  • center : les éléments sont placés au centre
  • space-between : les éléments sont séparés avec de l'éspace entre eux
  • space-around : les éléments sont séparés avec de l'éspace entre eux, mais il y'a aussi de l'éspace au début et à la fin
  • space-evenly : Les éléments sont répartis uniformément dans le conteneur d'alignement le long de l'axe principal.
  • stretch (par défaut) : les éléments s'étirent pour occuper tout l'éspace
