rowspan / colspan

"rowspan " est un attribute qui permet d'indiquer le nombre di lignes qu'un cellule <td> de tableau <table> peut couvrir.Cet attribut est utile si vous souhitez qu'une cellule s'etende verticalement sur plusieur lignes .Example si onmet rowspan="5", alors notre cellule occupera l'espase de 5 cellules verticalement . Cependat, il est important de noter que lorsq'on utilise cette manipulation , il faut retirer les td correspondants des autres lignes.

"colspan" est un attribut qui permet d'indiquer le nombre de colonnes qu’une cellule <td> de tableau <table> doit couvrir. Cet attribut est utile lorsque vous souhaitez qu’une cellule s’étende horizontalement sur plusieurs colonnes .Exemple si on met colspan="3", alors notre cellule occupera l'espace de 3 cellules horizontalement.
