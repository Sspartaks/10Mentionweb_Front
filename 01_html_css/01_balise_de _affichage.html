Le CSS dans la balise

La premiér façon de faire du style dans un élément est de déclarer l'attribut style sur celui-là. Ce n'est pas la meilleur maniére la faire

Je suis un paragraphe styléééééé !

Le CSS dans l'en-tete

Une seconde façon de faire du style sur notre page est de mettre la balise style dans la partie head. Cette méthode n'est toujours pas la plus conseillée mais elle permet plus de choses que la méthode balise en ligne, on peut désormais gràce à cette méthode viser plusieurs éléments d'un coup. Example, on met du ris en fond sur le corps de la page, les h2 en rouge et les h3 avec un fond rose

Le CSS dans une feuille de style externe

la derniére et meilleure façon pour faire du style est la méthode suivente : oncrée un dossier CSS avec un ficher style.css à l'intérieur il faudra alors lier nos deux fichers gràce à la balise link. On pourra alors faire du style sur les éléments aui se trouvent dans notre page. Example, ondéclare que tous les paragraphes doivent etre en bleu.
